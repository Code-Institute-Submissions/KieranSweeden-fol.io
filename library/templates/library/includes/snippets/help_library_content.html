The library is where you manage the creation of folios, editing their titles/descriptions, publishing/concealing them or deleting them entirely.

Folios are concealed by default when created. Assuming you have licenses available, you can publish a folio by clicking the Publish option within it's additional menu. Please be mindful that one license allows the publishment of one folio, which can later be swapped for another folio if you desire.

If you haven't bought any licenses yet, you can purchase some at the license store.

Legend

visibility  - folio is published.

visibility_off  - folio is concealed.
